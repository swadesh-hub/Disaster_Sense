Architecture Overview

This project simulates a fully-integrated disaster response network using state-of-the-art technologies.

React.js Django REST MySQL Maps API
  • Frontend SPA Architecture powered by modular components, responsive dashboard filters, and custom mapping.
  • Django Backend Architecture handles authentication, notification triggers, and relational schemas.
  • MySQL Relational Schema links reports to relief center allocations and logs audit details.

Disaster Management System

System Specifications & Tech Stack

Frontend (React.js SPA)

Built with styled components, responsive grid layouts, and modular view-routing. Handles location monitoring and real-time inputs.

Backend (Django REST Framework)

Manages 8 core models including Incident, ReliefCenter, EmergencyContact, and User Roles. Provides token-based JWT authentication.

Relational DB (MySQL)

Relational model mapping logs, incidents, relief center limits, and allocations via clean indexes and foreign constraints.

Map Integration (Maps API)

Interactive coordinate plotting for live disaster response vectors, dispatching squads, and safe zone calculations.
